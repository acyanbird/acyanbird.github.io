ubuntu virtual box 使用

发表于 2022-01-13 更新于 2022-07-18

这里记录一下重点，new 了

之后直接按照规矩走，然后设置里面剪贴板和拖拽开双向，存储这里开磁盘 https://.zhihu.com/p/42761302

哦我弄了 xp

遇到无法input的可在上面input 的 soft keyborad 选取

重启的时候等待！看到 press any boot from cd 的时候啥都别做

啊 key code 实在是找不到……所以记得装 VL 版本，因为这个是批量激活的所以网上的大部分都是这个版本的密钥

记得使用 setting-system-extended feature 的 enable APIC

为了让贡献发文件夹生效， devices 先选择最后一个 insert，会自动下载，重启一次打开我的电脑最下面 removable 安装，安装完毕会自动重启，若有问题参考这个

https://www.virtualbox.org/manual/ch04.html#additions-windows

然后在 my network places 选择 add a network place-choose another network location-browse-entire network-virtualbox—

然后就是你自己的文件夹啦，在此之前记得先设置，我设置的是transient folder，区别什么的不太清楚……

哦选择machine就会物理连接，记得选上 auto mount。然后就可以共享了

关于网络连接参考这个

https://itechbrand.com/virtualbox-windows-xp-no-internet/

需要安装一个驱动，目前似乎官方已经下架了……我扔github上

https://github.com/acyanbird/storing/blob/main/PRO2K3XP_x64.exe

64位的，下载安装重启即可。
