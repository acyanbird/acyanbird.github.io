kdenliveUsage

发表于 2022-07-01 更新于 2023-03-13

首先是导入，在 project bin 那边双击，选中视频就可以导入了。这次尝试降噪一下，使用audacity。那么右键这个视频 extract sound，然后audacity降噪就可，直接右键用audacity打开。

右键 track 可以 insert 或者 del，直接在最前最后点击可以 fade in fade out。

使用 razer tool，x,剪切

Kdenlive Linux Video Editing #8 Text and Image Overlay - YouTube

要遮挡一些东西，可以先 import 图片，然后把 clip property 拽出来，选择 position and zoom，调整大小就可以了。

Kdenlive Tutorial #4 | How to Text Overlay to Videos (Title Clip) - YouTube

然后是添加一点字，右键添加title。双击可以再次更改文字，记得点击 show background

kdenlive 声音处理 变速 音量增大减小 增益 - YouTube

选 volume 效果，然后调整大小就好
